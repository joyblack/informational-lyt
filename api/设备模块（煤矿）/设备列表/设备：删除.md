# 简介

# 访问地址
device-info/delete

# 请求参数

## 请求方式
POST

## 请求格式
JSON

## 请求数据
|参数名|类型|必填|说明|
|-|-|-|-|
|id|[number]|是|记录的ID|

## 请求示例
```json
{
	"id": 3
}
```

# 返回结果
**成功**
```json
{
    "state": true,
    "message": "操作成功",
    "detailMessage": "删除成功",
    "data": null,
    "code": 200
}
```

**失败**
```json
{
    "state": false,
    "message": "设备信息不存在",
    "detailMessage": "设备信息不存在",
    "data": null,
    "code": 113001
}
```
