# 结语
错误码具有如下的规律：
* 错误码由五位数构成，从100XX - 999XX变化。其中，前三位代表模块名，后两位代表错误编号。
* 通用错误码： 100XX
* 用户相关的错误码：101XX
* 部门相关的错误码：102XX
