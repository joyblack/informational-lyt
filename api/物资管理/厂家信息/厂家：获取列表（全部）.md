# 简介
获取列表。

# 访问地址
storehouse-manufacturer/getAllList

# 请求参数

## 请求方式
POST

## 请求格式
JSON

## 请求数据
|参数|类型|必填|简介|
|-|-|-|-|
|name|[string]||名称|
|code|[string]||编号|
|business|[string]||业务|
|contactPeople|[string]||联系人|
|contactPhone|[string]||联系电话|

## 请求示例
```json
{
    "name": "A"
}
```
