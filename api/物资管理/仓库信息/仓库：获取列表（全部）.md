# 简介
获取列表。

# 访问地址
storehouse-info/getAllList

# 请求参数

## 请求方式
POST

## 请求格式
JSON

## 请求数据
|参数|类型|必填|简介|
|-|-|-|-|
|name|[string]||仓库名称|
|responsePeople|[string]||负责人|
|admin|[string]||管理员|
|code|[string]||编号|
|isDefault|[enum]||是否为默认仓库：NO("否"),YES("是")|
|status|[enum]||仓库的状态：STOP("停用"),START("启用")|

## 请求示例
```json
{
	"name": "5号仓库",
	"status": "START",
    "responsePeople": "赵义",
    "code": "001-A",
    "admin": "Mr.Joy",
    "isDefault": "YES",
    "remarks":"aiyo,great!"
}
```
