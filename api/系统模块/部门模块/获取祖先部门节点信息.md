# 简介
返回当前节点的所有祖先节点的信息，同时包含自身的信息。

# 访问地址
department/getParentNodesJustIds

# 请求参数

## 请求方式
POST

## 请求格式
JSON

## 请求数据
|参数名|类型|必填|说明|
|-|-|-|-|
|id|[number]|是|记录ID|

## 请求示例
```json
{
	"id": 4
}
```
