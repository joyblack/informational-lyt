# 简介
根据当前账户所在的平台获取部门树，可一次性加载所有子部门信息。

注：若数据量过大，可采用 **获取子部门列表** 接口，一层一层的加载部门信息。

# 访问地址
server:port/information-dxn/v1/department/getCompanyTree

# 请求参数

## 请求方式
POST

## 请求格式
JSON

## 请求数据
无

## 请求示例
```json

```
