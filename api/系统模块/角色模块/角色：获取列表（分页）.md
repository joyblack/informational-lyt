# 简介
分页获取用户列表。

# 访问地址
system-role/getPagerList

# 请求参数

## 请求方式
POST

## 请求格式
JSON

## 请求数据
|参数名|类型|必填|说明|
|-|-|-|-|
|roleName|[string]||角色名称|
|permissions|[string]||权限字符串，资源列表串，用`-`分隔的ID字串|

## 请求示例
```json
{
	"roleName": "资料",
    "permissions": "1"
}
```
