# 3、数据格式
## 3.1、柱状图
很明显的对象数组，分别包含两个属性，代表列属性以及列值。
```javascript
 var data = [{
    year: '1951 年',
    sales: 38
  }, {
    year: '1952 年',
    sales: 52
  }
  ...
  }];
```

## 3.2、堆叠柱状图
堆叠柱状图以多个对象数组的形式形成多个组，每一组中k-v形式组织数据，其中，k=name的表名组名，其他的则是绘制数据。
```javascript
  var data = [{
    name: 'London',
    'Jan.': 18.9,
    'Feb.': 28.8,
    ...
  }, {
    name: 'Berlin',
    'Jan.': 12.4,
    'Feb.': 23.2,
    ...
  }];
```

## 3.3、分组柱状图
分组柱状图以name-k-v的形式组织数据。
```javascript
  var data = [{
    "name": "London",
    "月份": "Jan.",
    "月均降雨量": 18.9
}, {
    "name": "London",
    "月份": "Feb.",
    "月均降雨量": 28.8
}
...
}, {
    "name": "Berlin",
    "月份": "Jan.",
    "月均降雨量": 12.4
}, {
    "name": "Berlin",
    "月份": "Feb.",
    "月均降雨量": 23.2
}
...
];
```
